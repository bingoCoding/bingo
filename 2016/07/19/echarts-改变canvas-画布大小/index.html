echarts 改变canvas 画布大小

2016-07-19 · 56 words · 1 minute read

echarts canvas

canvas

在用echarts时，x轴显示过长的名字，会有重叠，或者显示不全问题，感觉画布框太小。通过调试发现，css定义在canvas里，用js调整总是不那么完美。经查资料发现要修改，so easy~。

x轴data名字太多，导致重叠或显示不全。在xAxis中加axisLabel。


xAxis : [ {
    axisLabel :{  
        interval:0,
        rotate: 60
    } 
} ]


interval:0 间隔，表示根据刻度个数进行调整。 如果你想要设置所有刻度均显示出来，则只需要设置当前属性值为0即可。

{string | number}interval ‘auto’类目型标签显示挑选间隔，默认为’auto’，可选为：’auto’（自动隐藏显示不下的） | ,0（全部显示） |{number}（用户指定选择间隔）

想要间隔多少个这里就可以配置多少的数值即可，根据个人需要作出相应配置。

rotate :60 倾斜角度，可以改变xAxis的data 数据名字倾斜角度。可以防止名字重叠。

倾斜之后，可能由于名字太长，导致显示不全，如上图。

调试发现 canvas 标签里有height 和style.height 两个height。通过改变两个height的高度，发现内层style.height默认是跟随外层height 进行缩放拉伸。两个height总是保持一致。不知道怎么调整两个的高度。后来发现，原来echarts已经提供了设置参数用来调节显示问题。

grid: { // 控制图的大小，调整下面这些值就可以，
     x: 40,
     x2: 100,
     y2: 115 // y2可以控制 X轴跟Zoom控件之间的间隔，避免以为倾斜后造成 label重叠到zoom上
 },

ok,问题解决。
